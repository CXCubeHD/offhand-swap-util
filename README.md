## Build

### Prerequisites

- Java 21 or higher
- Git

### Build Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/CXCubeHD/offhand-swap-util.git
   cd offhand-swap-util
   ```

2. **Build the mod**
   ```bash
   ./gradlew build
   ```

3. **Locate the JAR**
   The compiled mod will be available at:
   ```
   build/libs/offhand-swap-util-*.jar
   ```

4. **Install to Minecraft**
   - Place the JAR file in your `mods` folder
   - Launch Minecraft with Fabric Loader

### Development

To set up a development environment:

```bash
./gradlew genSources
```
